# 开发环境配置

开发环境的基本配置，记录用于备份。

## 介绍

由于经常切换Linux环境和部署项目，使得每次都要重新配置终端、配置环境和查找常用软件的安装步骤，十分繁琐，所以有了本项目。

## 包含

本项目包含```Django```和```Docker```的常用配置和国内安装配置源步骤，包含终端常用软件```zsh```和```vim```的便捷配置，具体请查看下表。
