# vim配置

## 配置项

```.vimrc
syntax on
set ts=4
set expandtab
set autoindent
set number
set relativenumber
```

配置详解
|序号|名称|功能|
|-|-|-|
|1|syntax||
|2|ts=s|tab=4空格|
|3|expandtab|会把一个```tab```字符替换成```tabsto(ts)```选项值对应的多个空格|
|4|autoindent|自动缩进|
|5|number|显示行号|
|6|relativenumber|显示相对行号|
